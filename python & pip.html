Python

python path/file.py Run a python file

python --version Display version of python

python --V Display version of python


Python Packages

pip install module name Install a python package

pip install module name==version Install exact version of a python package


pip show module name View version of a python package

pip index versions module name View all available versions of a python package

pip uninstall module name Uninstall a python package

Virtual Environment with Pipenv

pip install pipenv Install Pipenv

pipenv install package name Create virtual environment for a specific project


pip --venv View the virtual environment path

pipenv shell Activate virtual environment


pip install module name Install an additional package into the virtual environment
